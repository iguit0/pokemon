## Executar Projeto :computer: :wrench:


### :point_right: Clonando repositório
Clonando com SSH (recomendado)
```bash
git clone git@github.com:iguit0/pokemon.git
cd pokemon
```

Clonando com HTTPS
``` bash
git clone https://github.com/iguit0/pokemon.git
cd pokemon
```


### :point_right: Instalando dependências
Yarn (recomendado)
``` bash
yarn
```

NPM
``` bash
npm i
```

### :point_right: Modo de Desenvolvimento
``` bash
yarn dev
```

### :pushpin: Variáveis de ambiente
Para usar variáveis de ambiente tal como `REACT_APP_API_URL` deve ser necessário criar um arquivo `.env` seguindo o modelo de `.env.example`.
